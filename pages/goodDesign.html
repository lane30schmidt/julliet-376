The issue with the PS5 game bar is that it provides the user with no form of organization or convenience. If the user wants to move a game on the homescreen to make it more easily accessible, then they should be able to. Here is my proposed solution for allowing the user to move the games around.

Drag the PS5 logos back and forth between the different sections.





As the prototype shows above, instead of locking the user into the assigned order Sony has made, it would allow the user to move the games around the screen in any order they would like. This will allow for more customization and convenience for the user and better their experience while using the UI.
